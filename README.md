# **Employee Management System** 🏢✅  

A full-featured **Employee Management System** designed for efficient workforce and task management. It provides separate **Admin and Employee Dashboards**, ensuring smooth task assignment, tracking, and completion monitoring.  

---

## **🔹 Features**  

### **👨‍💼 Admin Dashboard**  
- View and manage employee details.  
- Assign tasks to employees.  
- Monitor real-time task progress.  
- Categorize tasks into **New, Active, Completed, and Failed**.  

### **👷 Employee Dashboard**  
- Accept new tasks assigned by the admin.  
- Track task progress in an organized manner.  
- Mark tasks as **Completed** or **Failed**.  
- Get a clear view of task status using a structured dashboard.  

### **🛠️ Technical Features**  
- **Context API** for centralized state management and smooth data flow.  
- **Tailwind CSS** for a fully responsive UI across different devices.  
- **Role-Based Access Control (RBAC)** to ensure admins and employees have distinct permissions.  

---

## **🚀 Tech Stack**  

- **Frontend**: React.js, Tailwind CSS  
- **State Management**: Context API  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Authentication**: JWT (JSON Web Tokens)  
- **Deployment**: Vercel (Frontend), Render (Backend)
